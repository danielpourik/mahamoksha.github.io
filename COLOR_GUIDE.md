## 🔄 مقایسه قبل و بعد

### هدر (Navbar)

**قبل:**
```
[ورود] [ثبت‌نام]     [میترا]     [☰]
┌──────────────────────────────────────┐
│ قهوه‌ای روشن + طلایی                 │
└──────────────────────────────────────┘
```

**بعد:**
```
[ورود] [ثبت‌نام]     [میترا]  [خانه] [دوره‌ها] [حساب] [درباره] [تماس]
┌────────────────────────────────────────────────────────────────┐
│ سبز جنگلی → فیروزه‌ای + طلایی (underline animation)           │
└────────────────────────────────────────────────────────────────┘
```

### رنگ‌های منوی:

**قبل:**
- متن: طلایی روشن
- Hover: سفید

**بعد:**
- متن: طلایی روشن
- Hover: طلایی روشن (بدون تغیر)
- Underline: حرکت از چپ به راست

## 🎯 استفاده رنگ‌ها در صفحات

### 1️⃣ هدر و ناوبری
```
Background: سبز عمیق → فیروزه‌ای (Gradient)
منوی متن: طلایی روشن
منوی Hover: طلایی روشن + Underline
دکمه ورود: Border طلایی، هوور → پر شود
دکمه ثبت‌نام: فون طلایی، هوور → روشن شود
لوگو: طلایی
```

### 2️⃣ بخش Hero
```
Background: سبز عمیق (Overlay)
عنوان: طلایی
دکمه‌ها: سفید با border طلایی
Hover: border قرمز، بالا بروند
```

### 3️⃣ دوره‌های آموزشی
```
Background: خاکی روشن
عنوان: سبز عمیق
Border چپ: طلایی
Hover Border: قرمز
دکمه: Gradient سبز → فیروزه‌ای
Hover دکمه: Gradient طلایی
```

### 4️⃣ فوتر
```
Background: سیاه-سبز
Border بالایی: طلایی
متن: طلایی روشن
Link: طلایی
Hover Link: قرمز
```

## 🎨 کمبوهای رنگی

### مثال 1: CTA Box
```css
border: 2px solid #d4af37;        /* طلایی */
background: white;
h3: #1a472a;                      /* سبز */
:hover border: #c9302c;           /* قرمز */
```

### مثال 2: دکمه
```css
background: linear-gradient(
    135deg,
    #1a472a 0%,        /* سبز */
    #0d4d3d 100%        /* فیروزه‌ای */
);
color: white;
:hover background: linear-gradient(
    135deg,
    #c9302c 0%,         /* قرمز */
    #1a472a 100%
);
```

## 🎨 پیشنهادات سفارشی‌سازی

اگر می‌خواهید رنگ‌ها را تغییر دهید:

### روش ساده:
1. `styles.css` باز کنید
2. بخش `:root` را پیدا کنید
3. رنگ‌ها را تغییر دهید:

```css
:root {
    --primary-color: #YOUR_COLOR;
    --secondary-color: #YOUR_COLOR;
    --accent-color: #YOUR_COLOR;
    /* ... */
}
```
