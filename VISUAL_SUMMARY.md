## 🎯 خصوصیات انیمیشن

### 1. Underline Animation
```
State 1: [خانه]     (بدون underline)
         ═════════════════

State 2: [خانه]     (با underline از چپ → راست)
         ═════════════════════════════
         
State 3: [خانه]     (کامل underline)
         ═════════════════════════════════════
```

### 2. Button Transform
```
State 1: [دکمه]     (عادی)

State 2: [دکمه]↗    (Hover - بالا رفتن)
         + Shadow بیشتر
```

### 3. Card Hover
```
State 1: ┌─────┐
         │ کارت │
         └─────┘

State 2:    ┌─────┐
           │ کارت │↗ (بالا رفتن 12px)
            └─────┘ (Shadow بیشتر)
```
