# گنجینه دانش‌های کهن آریایی - وبسایت

## توضیحات پروژه

یک وبسایت زیبا و حرفه‌ای برای انجمن دانش‌های کهن آریایی با طراحی مدرن و رابط کاربری دوست‌داشتنی.

## ویژگی‌های اصلی

### 1. **نوار ناوبری (Navigation Bar)**
- **سمت چپ**: دکمه‌های ورود و ثبت‌نام
- **وسط**: لوگو
- **سمت راست**: منوی همبرگر برای موبایل

### 2. **بخش Hero**
- تصویر پس‌زمینه با overlay
- عنوان اصلی: "گنجینه دانش‌های کهن آریایی راهنمای زندگی امروز ماست"
- دو کادر قابل کلیک:
  - **"به انجمن ما بپیوندید"**: برای عضویت
  - **"بیشتر بخوانید"**: برای مطالعه متن کامل

### 3. **مودال‌های (Modals)**
- **مودال عضویت**: فرم کامل با فیلدهای ضروری
- **مودال بیشتر بخوانید**: متن کامل متن معرفی
- **مودال ورود/ثبت‌نام**: صفحات جداگانه

### 4. **بخش دوره‌های آموزشی**
- شبکه کارتی (Grid) برای نمایش دوره‌ها
- هر کارت شامل:
  - تصویر دوره
  - عنوان
  - خلاصه
  - دکمه "بیشتر بخوانید"

### 5. **فوتر (Footer)**
- حقوق کپی رایت
- لینک‌های شبکه‌های اجتماعی

## چگونه استفاده کنید

### 1. **راه‌اندازی اولیه**
```bash
# باز کردن فایل HTML در مرورگر
open index.html
# یا
# فایل را به مرورگر بکشید
```

### 2. **استفاده از یک سرور محلی** (توصیه می‌شود)
```bash
# با Python 3
python -m http.server 8000

# یا با Python 2
python -m SimpleHTTPServer 8000

# یا با Node.js (اگر http-server نصب شده است)
http-server
```

سپس در مرورگر به `http://localhost:8000` بروید.

## تنظیمات تصاویر

تصاویر زیر را باید جایگزین کنید:

1. **`hero-bg.jpg`**: تصویر پس‌زمینه بخش Hero
   - اندازه توصیه‌شده: 1920×600px
   - فرمت: JPG/PNG

2. **`course1.jpg`، `course2.jpg`، `course3.jpg`، `course4.jpg`**: تصاویر دوره‌ها
   - اندازه توصیه‌شده: 400×300px
   - فرمت: JPG/PNG

### نحوه تغییر تصاویر

در فایل `index.html` خطوط زیر را پیدا کنید:

```html
<!-- برای hero section -->
<section class="hero" style="background-image: url('hero-bg.jpg');">

<!-- برای دوره‌ها -->
<div class="course-image" style="background-image: url('course1.jpg');"></div>
```

`hero-bg.jpg` و `course1.jpg` را با مسیر صحیح تصاویرتان جایگزین کنید.
